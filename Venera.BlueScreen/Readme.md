# Venera.BlueScreen
BlueScreen ist der Fehlerbildschirm von Venera. Er wird angezeigt, wenn ein Fehler auftritt, der nicht durch die Fehlerbehandlung von Venera abgefangen werden kann.
Er ist als CosmosOS-Plug geschrieben, um die Standard-Fehleranzeige (roter Text auf schwarzem Hintergrund) durch eine nutzerfreundliche und angenehme Alternative zu ersetzen.
Der BlueScreen kann jederzeit durch ein ENTER abgebrochen werden, was zu einem Neustart des Systems führt.
